A public, free-to-use meditation audio library built for deep rest, focus, and inner stillness. Project Frequency streams carefully selected ambient and frequency-style tones (like 432 Hz and 528 Hz) directly in the browser. 

Designed to be visually calming, simple to use, and entirely open‚Äîno logins, no tracking, just press play and breathe.

![Project Frequency Preview](https://via.placeholder.com/800x400.png?text=Project+Frequency)

## Features
- **Healing Frequencies:** Features specifically tailored soundscapes (432 Hz, 528 Hz, 963 Hz, and Low Theta/Rain).
- **Public & Free:** Built to be accessible to anyone with a browser. No sign-ups required.
- **Calm Interface:** A gentle, dark-mode gradient UI built with Tailwind CSS that is easy on the eyes, especially in dim lighting.
- **Fully Responsive:** Works perfectly on mobile phones, tablets, and desktop computers.

## Tech Stack
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Language:** TypeScript
- **Audio:** HTML5 Native Audio Streaming (Tracks sourced from Pixabay)

## üíª How to run locally (Option A)

If you'd like to download the code and run the website on your own computer, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation
1. Clone this repository to your computer (or download the ZIP file and extract it).
2. Open your terminal (or command prompt).
3. Navigate into the project folder.
4. Run the following command to install all the required dependencies:
   ```bash
   npm install
   ```

### Running the server
Once everything is installed, start the local development server:
```bash
npm run dev
```
Open your browser and visit `http://localhost:5173/` to see the site running live!

## üöÄ How to Deploy to the Internet (Free)

The easiest way to share this site with the world is by deploying it to **Vercel** using GitHub.

1. Create a free account on [GitHub](https://github.com/).
2. Create a new repository and upload all these project files (except the `node_modules` folder).
3. Create a free account on [Vercel](https://vercel.com/) and log in with your GitHub account.
4. On your Vercel dashboard, click **Add New** > **Project**.
5. Select the repository you just created and click **Import**.
6. Leave all the default settings and click **Deploy**.

Vercel will build your website and provide you with a live, public URL in less than a minute! Whenever you make updates to the code on GitHub, Vercel will automatically update your live site.
